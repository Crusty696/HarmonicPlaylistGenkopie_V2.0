# Quickstart

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the Application:**
    ```bash
    python main.py
    ```

3.  **Use the App:**
    *   Drag and drop your music folder onto the window.
    *   Wait for the analysis to complete.
    *   Export your sorted playlist as an `.m3u` file.
